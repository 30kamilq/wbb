Instrukcja instalacji spolszczenia dla WBB 3.1.x
1. Pobierz Default Avatar Extension: http://www.woltlab.com/pluginstore/plugin.html?pluginID=455
2. Zainstaluj pakiet
3. Zaimportuj plik pl.xml z archiwum w Style -> Języki -> Dodaj nowy język

Instrukcja instalacji spolszczenia dla WBB Lite 2.1
1. Pobierz Default Avatar Extension: http://www.woltlab.com/pluginstore/plugin.html?pluginID=455
2. Zainstaluj pakiet
3. Pakiet default_avatar.tar zaimportuj w System -> Pakiety -> Instaluj nowy pakiet
